# draw a retular hexagon正六变形 on the canvas画布 using a turtle 
```python
import turtle
turtle.right(-30)
turtle.fd(250)
turtle.right(60)
turtle.fd(200)
turtle.right(120)
turtle.fd(200)
turtle.right(60)
turtle.fd(200)
turtle.right(120)
```
## explination
```txt
# import turtle
import turtle module

# turtle.right(-30)
let turtle turn left 30 degrees

# turtle.right(60)
let turtle turn right 60 degrees

# turtle.fd(200)
let turtle move forward 200 pixels
```
